Go เป็นภาษาการเขียนโปรแกรมแบบโอเพ่นซอร์สที่พัฒนาโดย Google
โดยมุ่งเน้นไปที่:

การประมวลผลที่มีประสิทธิภาพ
การคอมไพล์ที่รวดเร็ว
ความง่ายในการเขียนโปรแกรม

เนื่องจากเหตุผลเหล่านี้ Go จึงมีความเร็วสูง อ่านและเขียนโค้ดได้ง่าย และมีน้ำหนักเบา
ซึ่งเหมาะสำหรับการสร้างซอฟต์แวร์ที่ต้องการประสิทธิภาพสูง

ในการเขียนโปรแกรม Go แรกของเรา เราจะเรียนรู้ไวยากรณ์พื้นฐานของ Go 
โดยเริ่มจากโปรแกรมง่ายๆ เช่นการพิมพ์ข้อความ "Hello, World!" 
เพื่อทำความคุ้นเคยกับวิธีการเขียนโค้ดและรันโค้ดในภาษา Go 
ซึ่งจะช่วยให้เราเริ่มต้นเข้าใจหลักการและโครงสร้างการเขียนโปรแกรมพื้นฐานในภาษา Go

ในโปรแกรม Go ทุกโปรแกรมจะเริ่มต้นด้วยฟังก์ชัน main() ซึ่งเป็น
ฟังก์ชันหลักที่ใช้ในการดำเนินการโปรแกรม
 
func main() {
    // โค้ดจะอยู่ในนี้
}

ก่อนใช้ฟังก์ชันนี้เราต้องนำเข้าแพ็กเกจ main โดยใช้คำสั่ง package main
หมายเหตุ: บรรทัดที่เริ่มด้วย // เป็นการเขียนคอมเมนต์ ซึ่งจะช่วยให้ผู้ใช้
งานเข้าใจโค้ด แต่คอมไพล์เลอร์ของ Go จะไม่สนใจ

ใน Go การพิมพ์ข้อความสามารถทำได้โดยใช้ฟังก์ชัน fmt.Println()
ซึ่งจะพิมพ์ข้อมูลภายในวงเล็บ () ลงบนหน้าจอ
ก่อนที่จะใช้ฟังก์ชัน fmt.Println() เราต้องนำเข้าแพ็กเกจ fmt โดยใช้คำสั่ง
import "fmt"

หลังจากนั้นเราสามารถพิมพ์ข้อความได้ เช่น:
fmt.Println("Hello, World!")

โครงสร้างพื้นฐานของโปรแกรม Go จะเริ่มจากฟังก์ชัน main() เสมอ
โดยทุกโปรแกรม Go ต้องมีโครงสร้างขั้นต่ำดังนี้

package main
func main() {
    // โค้ดจะอยู่ในนี้
}

ฟังก์ชัน main() เป็นจุดเริ่มต้นของการทำงานของโปรแกรม และทุกครั้ง
ที่เราจะใช้ฟังก์ชันอื่นๆ เช่น fmt.Println() จะต้องนำเข้าแพ็กเกจที่
เกี่ยวข้อง เช่น fmt

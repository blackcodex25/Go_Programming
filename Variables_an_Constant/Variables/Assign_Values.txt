ในการกำหนดค่าตัวแปรในภาษา Go มี 3 วิธีหลัก:
วิธีที่ 1
var number int = 10
กำหนดค่า 10 ให้กับตัวแปร number โดยระบุชนิดข้อมูล int

วิธีที่ 2
var number = 10
ไม่ระบุชนิดข้อมูลโดยตรง แต่คอมไพเลอร์จะตรวจสอบชนิดจากค่าที่กำหนด
(ในกรณีนี้คือ int)
กำหนดค่า 10 ให้กับตัวแปร number โดยไม่ระบุชนิดข้อมูล

วิธีที่ 3
number := 10
ใช้ตัวดำเนินการ := เป็นรูปแบบย่อในการกำหนดค่าให้ตัวแปร

หมายเหตุสำคัญเกี่ยวกับตัวแปรใน Go
1.หากตัวแปรไม่ได้รับการกำหนดค่าเริ่มต้น จะมีค่าเริ่มต้นถูกกำหนดให้
กับมันโดยอัตโนมัติ ตัวอย่างเช่น

var count int 
fmt.Println(count)

ในที่นี้ ตัวแปร count จะพิมพ์ค่า 0 (ค่าดีฟอลต์สำหรับ int) เนื่องจากเรา
ไม่ได้กำหนดค่าใดๆ ให้กับมัน

ใน Go ตัวแปรทุกตัวต้องมีประเภทข้อมูลที่เกี่ยวข้อง หากไม่มีจะทำให้โปรแกรมเกิด
ข้อผิดพลาด

// ข้อผิดพลาด: ตัวแปร count ไม่มีประเภทข้อมูล
var count

// ตัวแปร count1 เป็นประเภท int 
var count1 int

// ตัวแปร count2 เป็นประเภท int
var count2 = 10

กำหนดประเภทข้อมูลอย่างถูกต้องเป็นสิ่งสำคัญเพื่อให้โปรแกรม
ทำงานได้อย่างถูกต้องแและไม่มีข้อผิดพลาด

ในภาษา Go เราสามารถประกาศตัวแปรได้โดยใช้คำสั่ง var ตามด้วยชื่อและ
ชนิดของข้อมูล เช่น

var number int

รายละเอียด:

number คือชื่อตัวแปร
var คือคำสั่งที่ใช้ประกาศตัวแปร
int คือชนิดข้อมูลของตัวแปรนี้ (เป็นจำนวนเต็ม)

ตัวแปรใช้เพื่อเก็บข้อมูลที่สามารถนำไปใช้งานภายหลังในโค้ด

ในภาษา Go ตัวแปรทุกตัวจะมีชนิดข้อมูล (data type) ที่กำหนดไว้แบบ
ตายตัว ซึ่งข้อมูลจะกำหนดประเภทของข้อมูลที่ตัวแปรนั้นสามารถเก็บได้

ตัวอย่างเช่น:
var number int
ตัวแปร number ถูกกำหนดให้เป็นชนิด int ดังนั้นมันสามารถเก็บเฉพาะข้อมูลที่
เป็นจำนวนเต็มเท่านั้น
